# App UI Foundation

Drop these files into your React Native + Expo + NativeWind app.

Suggested placement:
- `src/theme/*`
- `src/components/ui/*`
- `src/components/patterns/*`
- `src/utils/cn.ts`

Designed for:
- React Native
- Expo
- TypeScript
- NativeWind

## Includes
- design tokens
- shadows
- reusable UI primitives
- shared screen patterns
